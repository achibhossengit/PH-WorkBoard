Question-1: What are the different ways to select an element in the DOM?

Way to select an element in DOM:

  1. getElementById()
  2. getElementsByClassName()
  3. querySelector()
  4. querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML returns Inner HTML including tags.

innerText returns Visible text only.

textContent returns All text inside element (visible or hidden).

Question-3: What is event delegation in the DOM?

It is the technique of adding a single event listener to a parent element to handle events triggered by its child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling means when you click or do something on an element, the event first happens there, then it moves up to that element’s parent, then to the parent’s parent, and so on. It’s like bubbles rising up through water.

Question-5: How do you create, add, and remove elements using JavaScript?

Create: document.createElement('tag')
Add: parent.appendChild(child)
Remove: parent.removeChild(child)
